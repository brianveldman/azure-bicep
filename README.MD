## Getting Started
To get started with Bicep you need the following
- Azure CLI Installed
```powershell
winget install -e --id Microsoft.AzureCLI
``` 
- Bicep CLI Installed
```powershell
winget install -e --id Microsoft.Bicep
``` 
- A IDE for example VS Code
```powershell
winget install -e --id Microsoft.VisualStudioCode
```
